# vue-demo-music

> 这是一个用Vue写的简单音乐demo


### 用到的技术
- vuejs2.x
- vue-resource
- Muse-UI(一个基于Vue2.x的material design 风格UI框架)

这个demo里没有用vuex进行数据通信，后面会用vuex重写这些数据的通信。

## 安装运行

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```
